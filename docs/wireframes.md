## 2) Park Profile

[Hero]
- Park name, rating count, save button
- Photo carousel

[Key facts]
- Age-fit badges (0-2, 3-5, 6-9, 10-12)
- Amenities icons (bathrooms, shade, fenced, water)
- Parking summary (free/paid, lot/street, walk mins)

[Community]
- Tips list (most recent first)
- "Report parking" and "Add tip" actions

- Age ranges (multi-select)
- Amenities (multi-select)
- Parking type (free/paid/lot/street)
- Distance radius slider

## 4) Plan Visit (lightweight)

- Date/time picker
- Notes
- Save to Favorites

## 5) Add Tip / Parking Report

- Tip text (short)
- Parking availability (low/med/high)
- Paid/free toggle
- Submit
