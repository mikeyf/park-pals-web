# Park Pals

Web PWA MVP for parents to find parks in Petah Tikvah. Map-based discovery with age-fit features, amenities, and parking info.

## Structure
- docs/: wireframes and seed data plan
- web/: frontend placeholder (MapLibre planned)

## Next steps
- Decide UI direction and confirm data fields
- Implement MapLibre map view + list view
- Build seed dataset for Petah Tikvah

## Run locally
- From `park-pals`, run: `python3 run.py`
- Open `http://localhost:8000` in your browser

## Deploy (GitHub Pages)
- Ensure changes are committed, then run: `scripts/deploy-pages.sh`
- Live URL: `https://mikeyf.github.io/park-pals-web/`
